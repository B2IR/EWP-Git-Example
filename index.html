Um an einem Projekt zu arbeiten, erstellt man sich zunächst eine lokale Kopie mit git clone. Die Dateien, an denen man Änderungen vorgenommen hat, fügt man mit git add einer Art Liste hinzu. Die so markierten Dateien müssen mit git commit lokal abgespeichert werden. Die neue Version kann dann mit git push auf den Server geladen werden. Mit git pull wird der lokale Klon mit dem Master synchronisiert.

Befehl Beispiel Beschreibung
$ git clone https://github.com/B2IR/EWP-Git-Example.git
$ git add index.html
$ git commit -m "Datum aktualisiert"
$ git push origin master
$ git pull https://github.com/B2IR/EWP-Git-Example.git
